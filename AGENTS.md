# Agents

本仓库用于长期记录技能基线、实践证据与能力升级，后续任何 AI Agent 或协作者在更新内容时都应遵守以下规则：

## 证据优先

- 不得把目标技能写成已掌握技能。
- 每个技能结论必须附带证据来源。
- 只有同时具备“具体场景 + 使用工具 + 个人动作 + 指标变化”时，才能升级技能等级。
- 只有理解概念，不能升级到 `能实践`。
- 只有参与项目，不能自动升级到 `能主导`。

## 脱敏表达

- 涉及公司项目时，只保留方法论、工具名、指标类型和收益区间。
- 不写敏感业务名称、内部参数、内部配置和不可公开实现细节。
- 能用抽象描述表达的内容，不写可识别内部上下文。

## 回写规则

- 新的 `log / benchmark / report` 必须注明它支撑了哪个技能升级。
- 技能升级必须同步回写到 `docs/skill-baseline.md`。
- `docs/profile.md` 只保留稳定的人物画像，不堆砌细碎过程。
